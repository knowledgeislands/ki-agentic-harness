# ki-skills

Audit, review, extract, and write Agent Skills against current best practice.

**Invoke:** `ki-skills audit <skill-or-repo> | conform <skill> | educate <description> | extract <repo> [--history <path>...] | help | optimise <skill> | refresh | review <skill-or-repo>`

**Modes:**

- `AUDIT` — review an existing skill
- `CONFORM` — bring an existing skill into line
- `EDUCATE` — write a new skill
- `EXTRACT` — identify reusable capabilities from a project
- `HELP` — explain this skill and stop; the default when no mode is given (then routes, if interactive)
- `OPTIMISE` — push a compliant skill toward excellent
- `REFRESH` — re-anchor best practice
- `REVIEW` — assess an existing skill's evolution opportunities

**See also:** `ki-agents` (subagent defs), `ki-mcp` (server code), `ki-authoring` (Markdown/TOML style), `ki-harness` (bundle layout).
